# MonRogue

PokéRogue 스타일의 몬스터 배틀 게임. 11마리 오리지널 몬스터, 4가지 타입(화염/해류/자연/정령), 상태이상 시스템, 잡기/교체/경험치 시스템 포함.

## 플레이

[https://harryyou1111-collab.github.io/monrogue](https://harryyou1111-collab.github.io/monrogue)

## 기능

- 11마리 오리지널 몬스터 (AI 이미지 생성 + 직접 디자인)
- 4가지 타입 상성 시스템
- 파티 시스템 (최대 3마리)
- 잡기 / 교체 / 레벨업 / 경험치
- 상태이상 4가지: 화상 / 마비 / 잠 / 독
- 코인 상점 + 아이템 뽑기
- 모바일 최적화 UI
